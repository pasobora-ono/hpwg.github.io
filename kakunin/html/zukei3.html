ヒント

配色

  1. 背景色にantiquewhite
  2. divに4種のbox-shadowを内側に設定
  3. 1番目に2.5emのchartreuse
  4. 2番目に2.5em ＋ 1pxのblack
  5. 3番目に5emのaqua
  6. 4番目に5em ＋ 1pxのblack
  7. 複数のbox-shadowは「, 」でつなげます。
    box-shadow: 0 0 0 2.5em chartreuse inset, 0 0 0 calc(2.5em + 1px) black inset, 0 0 0 5em aqua inset, 0 0 0 calc(5em + 1px) black inset;

テキストの上下中央揃え

  1. 「top:50%」でテキストの高さ含め中央に配置します。
  2. 「transform: translateY(-50%);」でテキストの高さ分だけ戻します。
